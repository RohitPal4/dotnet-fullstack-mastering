# Day 1 – C# Basics

## Topics Learned
- Data types (int, long, float, double, decimal)
- Type conversion (Parse, Convert, TryParse)
- String interpolation
- Formatting values
- String immutability and StringBuilder
- Null handling (null, ??, ?. operators)
- object vs dynamic

## Key Takeaways
- Prefer TryParse for safe conversions
- Use StringBuilder for frequent string changes
- Avoid dynamic unless necessary
- Null operators prevent runtime exceptions
